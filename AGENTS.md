## Project Rules

- Always place JavaScript in separate `.js` files under `src/js/`.
- Keep `src/js/main.js` as an import hub only; feature logic goes into dedicated files (e.g. `topbar.js`, `nav.js`).
- Use Tailwind CSS only.
- Do not write custom CSS unless absolutely necessary. If required, ask first. Place approved custom CSS in separate `.css` files under `src/css/`.
- Never run build commands. The user runs `npm run dev`, which handles builds automatically.
